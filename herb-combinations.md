# 常用对药与配伍禁忌

## 经方经典对药

| 对药 | 出处 | 配伍逻辑 |
|------|------|----------|
| **桂枝 + 芍药** | 桂枝汤 | 一散一收，调和营卫。等量（各3两）为和，桂多则散，芍多则敛 |
| **生姜 + 大枣** | 桂枝汤/小柴胡汤 | 姜辛散助卫，枣甘守助营。最平和的调和营卫对药 |
| **柴胡 + 黄芩** | 小柴胡汤 | 柴升清、芩降浊，一升一降疏利少阳 |
| **附子 + 干姜** | 四逆汤 | 回阳救逆，附子走而不守，干姜守而不走 |
| **石膏 + 知母** | 白虎汤 | 清热生津，膏清肺胃、母润肾燥 |
| **细辛 + 通草** | 当归四逆汤 | 散寒通脉，细辛温散、通草通利 |
| **吴茱萸 + 生姜** | 当归四逆加吴茱萸汤 | 温散肝胃久寒 |

## 张锡纯特色对药

| 对药 | 方剂 | 配伍逻辑 |
|------|------|----------|
| **山萸肉 + 知母** | 曲直汤 | 萸肉补肝收敛，知母清经络郁热——补中兼清 |
| **乳香 + 没药** | 活络效灵丹/曲直汤 | 活血止痛黄金搭档，乳香偏行气、没药偏散瘀 |
| **代赭石 + 牛膝** | 镇肝熄风汤 | 引血下行，赭石重镇、牛膝引经，一镇一引 |

## 下肢痹痛常用对药

| 对药 | 作用 | 偏向经络 |
|------|------|----------|
| **独活 + 桑寄生** | 祛风除湿+补肝肾 | 太阳膀胱经/少阴肾经 |
| **威灵仙 + 鸡血藤** | 行气通经+养血活血 | 通行十二经，对胆经侧线尤佳 |
| **川牛膝 + 杜仲** | 引血下行+强腰膝 | 下肢通用 |
| **地龙 + 全蝎** | 虫类通络 | 络脉，治麻木比疼痛更有效 |

## 治高血压常用药（非经方，六家常用）

| 药物 | 作用 | 适用条件 |
|------|------|----------|
| 川牛膝 | 引血下行 | 下肢痹痛+高血压配合使用，一箭双雕 |
| 天麻 | 平肝息风 | 舌红+头晕+高血压 |
| 钩藤 | 清热平肝 | 肝阳上亢+舌红 |
| 杜仲 | 补肝肾降压 | 虚性高血压 |
| 白芍（大量） | 酸甘化阴、柔肝 | 脉弦+高血压，20g以上 |

## 配伍禁忌速查

| 情况 | 禁用/慎用 | 风险 |
|------|-----------|------|
| 湿浊未清 | 熟地、山萸肉（大量）、阿胶 | 滋补助湿，关门留寇 |
| 舌苔黄腻 | 熟地、附子、肉桂 | 助热化燥 |
| 舌红少苔 | 麻黄、桂枝（大量）、细辛 | 伤阴动血 |
| 82岁以上 | 乳香、没药（大量）、甘遂、大戟 | 伤胃、伤正 |
| 脉浮大有力+舌淡 | 强力平肝药（赭石、龙骨、牡蛎） | 舌不红不支持肝阳上亢 |
| 有汗出 | 麻黄、大量桂枝 | 过汗亡阳 |
| 无汗出 | 桂枝汤为主方 | 桂枝本为解肌，无汗不可用 |
| 苔微黄+发热感 | 大量附子、肉桂、吴茱萸 | 助热 |
| 梅花鹿角类 | 高龄阴虚者 | 温燥伤阴 |
