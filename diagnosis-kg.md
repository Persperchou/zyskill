# 辨证知识图谱（Diagnosis Knowledge Graph）

> 静态知识规则库。每次临床辨证结束后，根据本图谱构建该病例的推理路径图。
> 本图谱涵盖：症状→病机→方证的层级推理关系、各家分歧节点、排除/禁忌规则。

## 二、第一层：症状→六经定位

### 2.1 六经病位速查

| 六经 | 典型症状定位 | 方证方向 |
|------|------------|----------|
| **太阳** | 项背、头、腰、下肢后侧、恶风寒、脉浮 | 桂枝汤/麻黄汤系列 |
| **阳明** | 前额、腹部、口渴、汗出、脉洪大 | 白虎汤/承气汤系列 |
| **少阳** | 身侧、胸胁、大腿外侧、口苦咽干、耳鸣、寒热往来 | 柴胡汤系列 |
| **太阴** | 腹部虚满、下利、食不下、时腹自痛 | 理中汤/四逆辈 |
| **少阴** | 但欲寐、手足寒、脉微细 | 四逆汤/真武汤 |
| **厥阴** | 四肢逆冷、消渴、气上撞心 | 乌梅丸/当归四逆汤 |

### 2.2 经络-症状-引经药三角

| 经络 | 大腿痛位置 | 挟症 | 引经药 |
|------|-----------|------|--------|
| 足太阳膀胱经 | 后侧（承扶→委中→承山） | 腰背强 | 葛根/独活 |
| 足少阳胆经 | **外侧**（环跳→风市→阳陵泉） | 耳鸣/眼痒/口苦 | **柴胡** |
| 足阳明胃经 | 前侧（髀关→伏兔→足三里） | 消谷善饥 | 白芷/葛根 |
| 足太阴脾经 | 内侧前 | 腹胀/泄泻 | 苍术 |
| 足厥阴肝经 | 内侧中 | 少腹拘急 | 川楝子 |
| 足少阴肾经 | 内侧后 | 腰酸/膝软 | 独活/桑寄生 |

## 三、第二层：脉舌→寒热虚实判定

### 3.1 脉诊判定路径

```
脉浮 → 病在表
   ├── 浮缓/浮弱 → 表虚（桂枝汤证） 
   ├── 浮紧 → 表实（麻黄汤证）
   └── 浮大有力 → ⚠️ 看年龄！
        ├── 青壮年 → 表实证
        ├── 82岁以上 → 须结合舌象，防虚阳外浮
        │   ├── 舌淡红+苔白 → 非热，考虑虚阳外浮
        │   └── 舌红+苔黄 → 真热
        └── 沉取 → 
            ├── 沉取有力 → 里实（湿瘀/食积）
            └── 沉取无力 → 真虚假实

脉沉 → 病在里
   ├── 沉细无力 → 阳虚（四逆汤类）
   ├── 沉弦 → 水饮/气滞
   └── 沉大有力 → 里实（湿瘀/痰饮/食积）
```

### 3.2 舌诊排除规则（核心知识）

| 规则 | 内容 | 应用场景 |
|------|------|----------|
| **R1 红舌排除阴寒** | 舌红≠可温补。舌红必须清热/滋阴 | 防真热假寒误辨 |
| **R2 淡舌排除肝阳上亢** | 舌淡红=不是肝阳上亢。即使血压高、头晕，不加天麻、钩藤、代赭石等平肝药 | 这是最常用的排除规则 |
| **R3 苔白滑排除阴虚** | 苔白滑=有湿。不能用熟地、阿胶、大量山萸肉 | 防滋补助湿 |
| **R4 苔黄排除纯寒** | 苔黄=有热（即使舌淡）。不能用大量附子、肉桂 | 防火上浇油 |
| **R5 舌鼓排除纯虚** | 舌体鼓胀=有湿浊。不能纯补，必须化湿为先 | 防关门留寇 |
| **R6 苔骤厚排除补益有效** | 服药后苔突然变厚腻=邪被兜住 | 立即调整方药 |

## 四、第三层：方证匹配

### 4.1 下肢痹痛方证决策树

```
下肢胀/痛/麻
├── 劳则发、静则止 → 虚证为主
│   ├── 酸胀+发热 → 肝虚湿热（曲直汤）
│   ├── 酸软无力+冷 → 肾阳虚（肾四味+附子）
│   └── 麻木+不仁 → 血痹（黄芪桂枝五物汤）
│
├── 持续存在、不因动静变化 → 实证为主
│   ├── 胀痛+红肿 → 湿热痹（四妙散）
│   ├── 刺痛+固定不移 → 血瘀（活络效灵丹）
│   └── 冷痛+畏寒 → 寒湿痹（乌头汤/当归四逆汤）
│
└── 混合型（虚+实）
    └── 胀痛+劳重+发热+舌不红 → 曲直汤加减（张锡纯）
```

### 6.2 关键知识节点（从本案提炼）

| 节点 | 内容 | 提炼来源 |
|------|------|----------|
| **KN-01** | 舌淡红 → 排除肝阳上亢，不加天麻/钩藤/代赭石 | 周老师纠正 |
| **KN-02** | 劳则发+静则止 → 虚证为主，实证不会因为坐下就消失 | 大儿童症状特征 |
| **KN-03** | 胀时发热感 → 不是纯寒证，是有郁热 | 大儿童补充描述 |
| **KN-04** | 山萸肉20g+湿未清 → 气机被遏→眼睛模糊/血压↑ | 服药后反应分析 |
| **KN-05** | 湿未清(舌鼓) → 不可加肾四味/熟地 | 周老师追问+安全判断 |
| **KN-06** | 耳朵/眼睛症状+大腿外侧 → 胆经一体，用小柴胡 | 经络辨证推理 |
| **KN-07** | 减量续服 → 等效日量减半即为安全剂量 | 剩余药处理方案 |

## 七、知识图谱的演化规则

1. 每完成一个病例的辨证，将**新的关键知识节点**追加到本文件
2. 若新知识节点与已有知识冲突，标注为 `[待验证]` 并记录冲突双方
3. 知识节点的置信度分级：
   - `[已确认]`：3个以上独立案例验证
   - `[待验证]`：仅有1-2个案例支持
   - `[SciClaw 推演]`：纯逻辑推导，无临床验证
4. 每年由周老师 review 一次知识节点的有效性
