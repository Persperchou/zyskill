> 静态知识规则库。每次临床辨证结束后，根据本图谱构建该病例的推理路径图。
> 本图谱涵盖：症状→病机→方证的层级推理关系、各家分歧节点、排除/禁忌规则。

## 二、第一层：症状→六经定位

### 2.1 六经病位速查

| 六经 | 典型症状定位 | 方证方向 |
|------|------------|----------|
| **太阳** | 项背、头、腰、下肢后侧、恶风寒、脉浮 | 桂枝汤/麻黄汤系列 |
| **阳明** | 前额、腹部、口渴、汗出、脉洪大 | 白虎汤/承气汤系列 |
| **少阳** | 身侧、胸胁、大腿外侧、口苦咽干、耳鸣、寒热往来 | 柴胡汤系列 |
| **太阴** | 腹部虚满、下利、食不下、时腹自痛 | 理中汤/四逆辈 |
| **少阴** | 但欲寐、手足寒、脉微细 | 四逆汤/真武汤 |
| **厥阴** | 四肢逆冷、消渴、气上撞心 | 乌梅丸/当归四逆汤 |

| 经络 | 大腿痛位置 | 挟症 | 引经药 |
|------|-----------|------|--------|
| 足太阳膀胱经 | 后侧（承扶→委中→承山） | 腰背强 | 葛根/独活 |
| 足少阳胆经 | **外侧**（环跳→风市→阳陵泉） | 耳鸣/眼痒/口苦 | **柴胡** |

| 足厥阴肝经 | 内侧中 | 少腹拘急 | 川楝子 |
| 足少阴肾经 | 内侧后 | 腰酸/膝软 | 独活/桑寄生 |

## 三、第二层：脉舌→寒热虚实判定

### 3.1 脉诊判定路径

```
脉浮 → 病在表
   ├── 浮缓/浮弱 → 表虚（桂枝汤证） 
   ├── 浮紧 → 表实（麻黄汤证）
   └── 浮大有力 → ⚠️ 看年龄！
        ├── 青壮年 → 表实证
        ├── 82岁以上 → 须结合舌象，防虚阳外浮
        │   ├── 舌淡红+苔白 → 非热，考虑虚阳外浮
        │   └── 舌红+苔黄 → 真热
        └── 沉取 → 
            ├── 沉取有力 → 里实（湿瘀/食积）
            └── 沉取无力 → 真虚假实

脉沉 → 病在里
   ├── 沉细无力 → 阳虚（四逆汤类）
   ├── 沉弦 → 水饮/气滞
   └── 沉大有力 → 里实（湿瘀/痰饮/食积）
```

### 3.2 舌诊排除规则（核心知识）

| 规则 | 内容 | 应用场景 |
|------|------|----------|
| **R1 红舌排除阴寒** | 舌红≠可温补。舌红必须清热/滋阴 | 防真热假寒误辨 |
| **R2 淡舌排除肝阳上亢** | 舌淡红=不是肝阳上亢。即使血压高、头晕，不加天麻、钩藤、代赭石等平肝药 | 这是最常用的排除规则 |
| **R3 苔白滑排除阴虚** | 苔白滑=有湿。不能用熟地、阿胶、大量山萸肉 | 防滋补助湿 |
| **R4 苔黄排除纯寒** | 苔黄=有热（即使舌淡）。不能用大量附子、肉桂 | 防火上浇油 |
| **R5 舌鼓排除纯虚** | 舌体鼓胀=有湿浊。不能纯补，必须化湿为先 | 防关门留寇 |
| **R6 苔骤厚排除补益有效** | 服药后苔突然变厚腻=邪被兜住 | 立即调整方药 |

## 四、第三层：方证匹配

### 4.1 下肢痹痛方证决策树

```
下肢胀/痛/麻
├── 劳则发、静则止 → 虚证为主
│   ├── 酸胀+发热 → 肝虚湿热（曲直汤）
│   ├── 酸软无力+冷 → 肾阳虚（肾四味+附子）
│   └── 麻木+不仁 → 血痹（黄芪桂枝五物汤）
│
├── 持续存在、不因动静变化 → 实证为主
│   ├── 胀痛+红肿 → 湿热痹（四妙散）
│   ├── 刺痛+固定不移 → 血瘀（活络效灵丹）
│   └── 冷痛+畏寒 → 寒湿痹（乌头汤/当归四逆汤）
│
└── 混合型（虚+实）
    └── 胀痛+劳重+发热+舌不红 → 曲直汤加减（张锡纯）
```

| **KN-01** | 舌淡红 → 排除肝阳上亢，不加天麻/钩藤/代赭石 | 周老师纠正 |
| **KN-02** | 劳则发+静则止 → 虚证为主，实证不会因为坐下就消失 | 大儿童症状特征 |
| **KN-03** | 胀时发热感 → 不是纯寒证，是有郁热 | 大儿童补充描述 |

| **KN-05** | 湿未清(舌鼓) → 不可加肾四味/熟地 | 周老师追问+安全判断 |
| **KN-06** | 耳朵/眼睛症状+大腿外侧 → 胆经一体，用小柴胡 | 经络辨证推理 |
| **KN-07** | 减量续服 → 等效日量减半即为安全剂量 | 剩余药处理方案 |
| **KN-08** | 脾虚→生湿→痰湿流注胆腑→息肉；脾湿→上蒙鼻窍→湿郁化热→黄涕。黄涕≠独立上焦热，可能是湿郁化热 | persper 案：周老师纠正 |

| **KN-10** | 接到症状第一时间不是辨证而是存疑：症状是否描述清楚？关联症状是否遗漏？需要补充信息吗？——B0 五问是辨证的前置条件 | 方法论：周老师新增 |

## 八、辨证方法论铁律（方法论层级，优先于具体规则）

> ⚠️ 以下规则源于实际辨证中的错误教训，每次辨证必须逐条执行。

### 铁律 1：多假说比对——禁止第一直觉锁定

**症状读完后的第一步，不是"这是什么证"——而是"这可能是哪几种证？"**

```
错误示范：
  读完四诊 → "上热下寒！" → 找证据支持上热下寒 → 开乌梅丸

正确流程：
  读完四诊 → 列出 ≥2 个竞争性假说 →
  ├── 假说 A：上热下寒（乌梅丸方向）
  |    证据：黄涕=上热，脚冷便溏=下寒
  |    反证：舌淡红≠热证舌，黄涕可能是湿郁化热
  ├── 假说 B：脾虚生湿（六君子汤方向）
  |    证据：齿痕+舌胖+便溏=脾虚湿盛铁证
  |    反证：[待找]
  └── 假说 C：... 
  → 比对证据支持度 → 收窄 → 选定
```

### 铁律 2：寻找统一病根——避免拼盘诊断

**如果诊断需要多个独立假设来解释不同症状，优先寻找能统一所有症状的单一病根。**

```
拼盘诊断（差）： 上焦有热 + 中焦脾虚 + 下焦肾阳虚 → 三路用药
统一诊断（好）： 脾虚生湿 → 湿郁化热(上) + 湿流注胆(中) + 湿伤阳气(下) → 一路健脾
```

### 铁律 3：六家分析要展示推理链，不是展示结论

**每条医家的分析必须包含"因为……所以……"的完整推理，不能只是"定位厥阴→乌梅丸"。**

```
错误（只给结论）：
  胡希恕：此属厥阴病，乌梅丸主之。

正确（展示推理）：
  胡希恕：
  1. 六经筛选：太阳？脉不浮紧、无头项强痛→排除。
     阳明？无大渴大汗大热脉洪→排除。
     少阳？有胆府病变但无口苦咽干寒热往来→排除主证。
     太阴？大便不成形（下利）→太阴框架成立。
     厥阴？久利+上（鼻窍）有热象+下（脚冷）有寒象→
     太阴不足以解释上部的热，需要厥阴的寒热错杂框架。
     结论：太阴为基，厥阴为变，太阴厥阴并病。
  2. 方证比对：...（展开）...
```

### 铁律 4：舌诊优先——舌是辨证的定盘星

**当症状解读出现分歧时，以舌象为最高裁决依据。** 舌淡红推翻热证诊断的优先级高于黄涕支持热证诊断的优先级。

### 铁律 5：先存疑后辨证——症状信息的完整性是第一道关

**接到症状描述的第一时间不是辨证，是存疑。** 必须逐条追问：

```
存疑五问（详见 SKILL.md B0）：
1. 症状是否描述清楚？（部位？时间？诱因？持续？加重/缓解？）
2. 是否有相关联的症状未描述？（根据已有症状反推关联症，主动提问）
3. 患者是否理解其症状的全部维度？（患者可能遗漏自认为不重要的症状）
4. 现有信息能否排除关键鉴别诊断？（明确列出无法排除的方向）
5. 是否需要补充信息后再辨证？（关键信息缺失→先问，不可绕过）
```

```
错误（信息不足直接辨证）：
  患者：大便不成形、脚冷。
  → 直接判断：脾肾阳虚，四逆汤。
  → 漏了：大便不成形多久了？是水样还是溏软？有无腹痛？
    脚冷是全天冷还是冬天冷？手冷不冷？夜尿几次？

正确（先存疑）：
  患者：大便不成形、脚冷。
  → 🟡 存疑先行：脾肾阳虚方向成立，但缺以下信息——
    [待确认] 大便性状（水样/溏软）、频次、腹痛否？
    [待确认] 脚冷范围、昼夜差异、手温？
    [待确认] 小便频次、夜尿？
  → 先补充后方可收窄诊断。
```

### 铁律 6：开方必须考量患者三因素 + 六家方药辩论

**剂量大小和药性强弱，必须结合患者年龄、精神状态、脾胃运化能力三项因素共同决定。** 六家所开方药必须在 B2.5 环节展开交叉辩论：剂量为何不同？药性走向为何分歧？安全风险各自在哪？

**三因素剂量调整表：**

| 因素 | 评估标准 | 对剂量的影响 |
|------|----------|------------|
| **年龄** | 青年(<40)→用常量；中年(40-65)→常量偏轻；老年(65-80)→轻量；高龄(80+)→最小有效量 | 每升一档，攻伐药减20-30%，猛药减半 |
| **精神状态** | 健硕有力→耐受常规量；面色偏白/语声低微→减量；萎靡不振→减量+加强扶正 | 面色和语声是脾胃之气的外在表现 |
| **脾胃运化** | 纳可/大便成形→常规；纳差/便溏→减攻伐药+加护胃药 | 齿痕舌、关脉弱=脾胃弱的铁证 |

**六家方药辩论流程：**

```
剂量辩论：
  胡希恕开A方 附子6g → 基于"有是证用是药"，病轻药轻
  李可开A方 附子15g → 基于"阳虚重，非重剂不起"
  争议焦点：脉象和舌象是否支持"阳虚重"？
  裁决：尺弱但沉取有力 → 非极虚 → 胡希恕用量更合理

药性辩论：
  张锡纯方向（山萸肉酸收） vs 胡希恕方向（桂枝辛通）
  争议焦点：是否需要收敛？会不会留邪？
  裁决：患者有早泄 → 张锡纯方向合理，但须防酸收碍湿
  → 山萸肉仅用12g + 加茯苓20g化湿 = 收中有通
```

### 铁律 7：舌脉交叉比对——四诊合参必须从舌脉开始

**舌诊和脉诊是四诊中仅有的两个客观体征，必须在每次辨证中单独拿出来做交叉比对，不可混在四诊里一带而过。**

```
比对流程：
1. 逐项拆解舌象结论（寒热/虚实/病位/津液）
2. 逐项拆解脉象结论（表里/寒热/虚实/脏腑）
3. 逐维度交叉比对，标注一致/矛盾/互补
4. 一致的维度 = 强证据，矛盾 = 重新审查
5. 给出综合裁决，各家推演时必须引用此结论
```

| 舌脉比对结果 | 含义 | 行动 |
|------------|------|------|
| 🟢 一致（如舌淡+脉弱） | 表里深浅统一，可信度高 | 直接推进 |
| 🔴 矛盾（如舌红+脉沉弱） | 表里不一，真寒假热/真热假寒可能 | 暂停，重新审查 |
| 🟡 一虚一实（如舌淡胖+脉弦有力） | 本虚标实 | 标本先后必须分清 |
| 🟡 同向异度（如舌淡+脉略弱） | 方向一致力度稍不同 | 微调用量即可 |

```
错误（舌脉混在四诊里不单独比对）：
  舌淡红/苔白+脉沉弱 → 直接进入"脾肾阳虚"辨证
  → 看不出舌和脉各自说了什么、有没有矛盾

正确（单独比对）：
  舌诊结论：寒（苔白）+ 虚（淡红/齿痕）+ 病在脾（舌中裂纹）
  脉诊结论：里（沉）+ 虚（弱）+ 病在肾（尺弱）
  比对：舌主脾、脉主肾 → 互补不矛盾 → 脾肾同病 ✅
       舌虚=脉虚 → 虚证一致 ✅
       舌寒=脉不热 → 寒证一致 ✅
       → 全绿，可直接推进至脾肾双补方向
```
