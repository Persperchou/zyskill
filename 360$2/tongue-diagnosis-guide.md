# 舌诊辨证速查表

## 舌诊基础

| 舌部分区 | 对应脏腑 |
|----------|----------|
| 舌尖 | 心肺 |
| 舌中 | 脾胃 |
| 舌根 | 肾、膀胱 |
| 舌边 | 肝胆 |

## 舌质速查

| 舌质 | 主病 | 与伤寒论常见证对应 |
|------|------|------------------|
| 淡红 | 正常/表证/轻证 | 太阳病初起（桂枝汤/麻黄汤证） |
| 淡白 | 血虚/阳虚 | 当归四逆汤证、黄芪建中汤证 |
| 红 | 热证（实热/虚热） | 白虎汤证、栀子豉汤证 |
| 绛 | 热入营血 | 清营汤证 |
| 紫暗/瘀点 | 瘀血 | 桂枝茯苓丸证、抵当汤证 |
| 胖大/齿痕 | 气虚/阳虚湿盛 | 苓桂术甘汤证、真武汤证 |
| 瘦小 | 阴虚 | 炙甘草汤证、黄连阿胶汤证 |

## 舌苔速查

| 舌苔 | 主病 | 对应经方 |
|------|------|----------|
| 薄白 | 正常/表寒 | 桂枝汤、麻黄汤 |
| 薄白润 | 表寒湿 | 桂枝加附子汤 |
| 薄白干 | 表寒化热 | 桂枝二越婢一汤 |
| 薄黄 | 里热初起 | 小柴胡汤加石膏 |
| 黄燥 | 阳明实热 | 白虎汤、承气汤 |
| 黄腻 | 湿热 | 半夏泻心汤、茵陈蒿汤 |
| 白腻 | 寒湿/痰饮 | 苓桂术甘汤、小青龙汤 |
| 白滑 | 里寒湿盛 | 真武汤、四逆汤 |
| 灰黑而干 | 热极 | 大承气汤 |
| 灰黑而润 | 寒极 | 四逆汤、通脉四逆汤 |
| 少苔/无苔 | 阴虚 | 黄连阿胶汤、炙甘草汤 |
| 舌苔骤厚 | 邪气被遏 | 须检查是否敛邪/补益过早 |

## 常见证候的舌象速查

| 证候 | 舌质 | 舌苔 | 关键鉴别点 |
|------|------|------|----------|
| 太阳中风（桂枝汤） | 淡红 | 薄白润 | 舌不红是排除里热的关键 |
| 太阳伤寒（麻黄汤） | 淡红 | 薄白偏干 | 无汗→舌不如桂枝汤证润 |
| 阳明经证（白虎汤） | 红 | 黄燥 | 舌红绛+苔黄燥是铁证 |
| 少阳证（小柴胡汤） | 淡红/偏红 | 薄白或薄黄 | 舌边可能偏红 |
| 真寒假热（四逆汤） | 淡白胖嫩 | 白滑 | 舌不红是排除真热的铁证 |
| 真热假寒（白虎汤） | 红绛 | 黄燥/焦黑 | 四肢虽冷，舌必红绛 |
| 湿热痹阻 | 红 | 黄腻 | 局部发热+舌红苔黄腻 |
| 寒湿痹阻 | 淡白/淡红 | 白滑/白腻 | 无热感+舌淡苔白 |
| 肝阳上亢 | 红 | 薄黄/少苔 | 舌红是必要证据，不红则否定 |
| 血瘀证 | 紫暗/瘀点 | 可正常 | 舌下静脉怒张是辅助证据 |

## ⚠️ 关键排除法则

- **舌淡红基本排除肝阳上亢**——这是最常用的排除法
- **舌红排除真寒假热**（四逆汤证舌必不红）
- **舌淡白排除阳明实热**（白虎/承气证舌必红绛）
- **苔白滑排除阴虚证**（阴虚苔少或无或干黄）
- **苔骤厚腻排除补益有效**（应先化湿再议补）
