# 七家医家代表方剂

## 张仲景经方（六家共用基方）

### 桂枝汤类

| 方名 | 组成 | 桂林古本剂量参考 |
|------|------|----------------|
| **桂枝汤** | 桂枝、芍药、生姜、大枣、甘草 | 桂3两 芍3两 姜3两 枣12枚 草2两 |
| 桂枝加葛根汤 | +葛根4两，桂芍减至各2两 | 葛4 桂2 芍2 姜3 枣12 草2 |
| 桂枝加附子汤 | +附子1枚 | 桂3 芍3 姜3 枣12 草2 附1 |
| 桂枝去芍药汤 | -芍药 | 桂3 姜3 枣12 草2 |
| 黄芪桂枝五物汤 | -甘草，+黄芪 | 芪3 桂3 芍3 姜6 枣12（《金匮》） |

## 张锡纯独创方剂

### 曲直汤
**主治**：肝虚腿疼，左部脉微弱者。经络郁热者亦可用。

| 药物 | 原方用量 |
|------|----------|
| 山萸肉（净） | 一两（30g）——君药 |
| 知母 | 六钱（18g） |
| 乳香 | 三钱（9g） |
| 没药 | 三钱（9g） |
| 当归 | 三钱（9g） |
| 丹参 | 三钱（9g） |

### 活络效灵丹
**主治**：气血凝滞，经络瘀阻，心腹肢体疼痛。

| 药物 | 原方用量 |
|------|----------|
| 当归 | 五钱 |
| 丹参 | 五钱 |
| 乳香 | 五钱 |
| 没药 | 五钱 |

### 镇肝熄风汤
**主治**：肝阳上亢，头目眩晕，脉弦长有力。

| 核心药物 | 
|----------|
| 代赭石、龙骨、牡蛎、龟板、白芍、玄参、天冬、川楝子、麦芽、茵陈、甘草、牛膝 |

## 李可常用方剂/药物组合

### 肾四味
**适应症**：老年肝肾不足——腰膝酸软、夜尿多、下肢无力

| 药物 | 常用量 |
|------|--------|
| 枸杞子 | 15-30g |
| 菟丝子 | 15-30g |
| 盐补骨脂 | 15-30g |
| 仙灵脾 | 15-30g |

### 止痉散
**适应症**：顽固性麻木、疼痛、经络不通

| 药物 | 用法 |
|------|------|
| 全蝎 | 研末冲服 |
| 蜈蚣 | 研末冲服 |

### 破格救心汤
**适应症**：心衰、休克、真寒假热危重症

核心：大剂附子+山萸肉+龙骨牡蛎+人参+麝香

## 六家用药禁忌速查

| 医家 | 慎用/禁用 | 原因 |
|------|-----------|------|
| 胡希恕 | 桂枝汤用于无汗者 | 桂枝本为解肌，无汗不可用 |
| 柯琴 | 芍药用于胸满者 | 芍药酸寒，邪气未解时敛邪 |
| 郭生白 | 高血压大量用桂枝 | 辛温升散，可能升压 |
| 彭子益 | 表邪未解用补益 | 关门留寇 |
| 张锡纯 | 乳没量大伤胃 | 82岁以上需减量并配护胃药 |
| 李可 | 附子用于真热假寒 | 误用则死 |
